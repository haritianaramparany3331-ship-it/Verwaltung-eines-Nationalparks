This is a desktop app to manage the staff and the animals in a national parc.
I used C++ to implement the application logic and the Qt Framework to design the GUI.
The GUI (in NationalparkGUI.pro) layer and the logic (NationalPark.cpp.pro) layer are parted and are only connected with an object of a logic class in the GUI class.
The build system is qmake and I used CSS to design the main window.
